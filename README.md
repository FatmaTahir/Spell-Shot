Spell-Shot is an interactive web-based game designed to challenge players to guess hidden words based on hints provided. Developed entirely with HTML, CSS, and JavaScript, the game offers a smooth,<br>  and visually engaging experience suitable for users of all ages. It is structured to allow players to engage in five rounds per game session, with each round presenting a new word and corresponding clue.
<br>Upon launching the game, users are greeted with a start screen where they can choose their preferred difficulty level. <br>Each difficulty level corresponds to a predefined set of words with varying complexity. Once the game begins, players are shown a hint for the selected word and can start guessing by typing one letter at a time. The interface dynamically updates the display based on correct or incorrect guesses, and the score is updated in real-time. Players continue playing until five rounds are completed, after which a short delay is followed by a “Game Over” alert indicating the end of the session.

The design is clean and interactive. The interface elements, transitions, and score tracking provide a user-friendly experience without any need for external libraries or backend integration. All game logic is handled through JavaScript, ensuring lightweight performance and ease of customization. 

 It demonstrates core JavaScript concepts such as DOM manipulation, event handling, functions, arrays, conditionals, timers, and basic game logic. It also highlights the importance of clean UI design using CSS and semantic HTML structure for accessibility and maintainability.
